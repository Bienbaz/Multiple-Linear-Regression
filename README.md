# Multiple Linear Regression – Multi-Channel Marketing Analysis

## Project Overview
This project builds a **Multiple Linear Regression** model to analyze how TV, Radio, and Social Media advertising spends impact Sales. The analysis includes multicollinearity checking (VIF), model fitting with `statsmodels`, assumption validation, and business-oriented interpretation for marketing budget optimization.

## Dataset
- **File**: `marketing_sales_data.csv`
- Variables: TV (categorical), Radio, Social Media, Influencer (categorical), Sales
